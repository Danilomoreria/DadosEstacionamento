# Sistema de estacionamento

Projeto desenvolvido para estudo de Java, JDBC, SQL e Banco de Dados.

# Objetivo

Sistema Desenvolvido para controlar a entrada e saída de veículos em um estacionamento, registrando informações e calculando o valor da permanência.

Tecnologias Utilizadas

* Java
* JDBC
* MySQL
* SQL
* Git
* GitHub
* Eclipse IDE

# Funcionalidades

* Cadastro de veículos
* Registro de entrada
* Registro de saída
* Consulta de veículos
* Integração com banco de dados MySQL

# Estrutura do Projeto

```text
src/
├── model/
├── dao/
├── util/
└── Main.java
```
